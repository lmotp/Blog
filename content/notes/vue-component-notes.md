# Vue 컴포넌트 메모

컴포넌트는 입력, 출력, 조합이라는 세 가지 관점으로 보면 이해가 쉽다.

## 구조

- `props`: 바깥에서 들어오는 값
- `emit`: 바깥으로 내보내는 이벤트
- `slot`: 바깥에서 채워 넣는 영역

## 운영 팁

기능이 커질수록 단일 컴포넌트 안에서 다 처리하지 말고, 표시용과 컨테이너를 분리한다.
