# JavaScript 반응성 메모

프레임워크 반응성을 이해하려면 먼저 참조와 복사를 구분해야 한다.

## 기억할 점

- 객체와 배열은 참조 단위로 공유된다.
- 얕은 복사와 깊은 복사는 영향 범위가 다르다.
- 상태를 직접 바꾸는 지점과 파생 값을 계산하는 지점을 나눈다.

## 실전 기준

컴포넌트에서 바뀌는 값이 많아질수록, 원본 상태와 표시용 상태를 분리하는 편이 낫다.
